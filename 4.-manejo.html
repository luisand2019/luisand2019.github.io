4. Manejo

Instrucciones para el uso del CD
El presente curso esta compuesto de los siguientes acápites:

  • 1. Inicio= La pagina principal ó carátula del CD.

  • 2. Prologo = Se describen los objetivos motivadores del curso.

  • 3. Del Autor= El autor y sus temas de investigación.

  • 4. Manejo= Indicaciones para el manejo y uso del CD.

  • 5. Teoria= Acceso a los Temas del curso.

  • 6. Practica= Algunas practicas y ejercicios propuestos.

  • 7. De Interes= Materiales informáticos interesantes.

  • 8. Enlaces= Enlaces o links de Internet recomendados.


Para el correcto funcionamiento del CD se recomienda como minimo una resolución de 800 x 600 píxeles y como navegador Internet Explorer 6 o superior. De preferencia Chrome o Firefox actualizados. Internet no es necesario para la mayor parte del CD, sin embargo para algunos de los enlaces, si es necesario. Por ejemplo para el material online.

Para poder acceder a algunos de los documentos necesitaran tener instalados algunos programas como FLASH ó ACROBAT READER. Todos los equipos modernos cuentan con los mismos. Sin embargo a veces se debe contar con sus ultimas actualizaciones. Las cuales puede instalarlas desde estos enlaces:


  • Flash de aqui ===> https://get.adobe.com/es/flashplayer/  
  • Acrobat Reader de aqui ===> https://get.adobe.com/es/reader/



   Nota: necesita tener internet para que estos enlaces funcionen.


Nota 1: Si esta usando Windows XP ó superior,  (que cuenta con nuevas medidas de protección para evitar navegar en paginas Web que pudiesen ser "agresivas" con nuestra computadora), le aparecerán algunas alarmas de seguridad:

En algun caso deberá colocar el Mouse en la barra de advertencia y Activar o seleccionar la opción Permitir o habilitar contenido bloqueado como se observa a continuación (con el botón izquierdo y luego derecho) si quiere que todas las opciones del curso funcionen.



Incluso usando Windows 7 ó Internet explorer 9 le aparecera esa pregunta, a la cual debes confirmar su permiso de acceso o habilitarlo.


Nota 2: En el caso de que quiera ejecutar un programa (en este CD tenemos algunos), primero debe Descargar el archivo. Para ello le aparecera el siguiente aviso:


Al que debe confirmar que si quiere Descargar


Luego una vez bajado el archivo debe ejecutarlo de la siguiente manera, oprimiendo el boton izquierdo y eligiendo Abrir.





Con ello lograra que el programa sea ejecutado.



En el caso de que le pregunte o le salga un aviso preguntando si esta seguro de ejecutarlo cada vez:


Ahí puede eligir una de las siguientes 2 opciones:

  1. Permitir ahora: Si hacemos clic en este botón, pues hasta que cerremos el navegador, en la página web que hayamos permitido la ejecución, este funcionará. Pero la próxima vez que volvamos a acceder, nos volverá a salir el aviso y habrá que repetir esto.
  2. Permitir y recordar: Similar a la anterior, pero en este caso NO volverá a mostrar advertencias y funcionará siempre.

Esto no les traerá ningún problema mientras lo haga en webs seguras o conocidas.

El correcto funcionamiento de los programas ha sido testeado con Windows 98, Windows XP, 7, 8 y Windows 10, Internet explorer 6, 7, 9, 11, Firefox y Chrome. Sin embargo si se hallan fallas favor reportarlas a Luisand@hotmail.com.
